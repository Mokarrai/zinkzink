# zinkzink — Boş Proje İskeleti

Bu depo için minimal bir boş proje iskeleti.

Klasör yapısı önerisi
- src/       : Kaynak kodlar
- tests/     : Testler
- docs/      : Dökümantasyon

Hızlı başlama
1. Dosyaları oluştur:
   git init
   git add .
   git commit -m "İlk iskelet"
2. Uzaktan depoya bağla ve gönder:
   git remote add origin <REMOTE_URL>
   git branch -M main
   git push -u origin main

Notlar
- Bu iskelet dil bağımsızdır — dil tercihine göre `src/` içine örnek dosyalar ekleyebilirim (ör. `main.go`, `index.js`, `app.py`).
- Lisans, .gitignore gibi temel dosyalar eklendi.
